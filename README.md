# 📊 Portfolio Optimization with R

## 📌 Description
This project builds an optimal investment portfolio using Modern Portfolio Theory (Markowitz) and Monte Carlo simulation.

## 📊 Assets
- AAPL
- MSFT
- NVDA
- AMZN
- JPM

## ⚙️ Methodology
- Data from Yahoo Finance using quantmod
- Logarithmic returns
- Monte Carlo simulation (5000 portfolios)
- Optimization via Sharpe Ratio
